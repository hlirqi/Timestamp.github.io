# 月历软件

一个简单、美观、响应式的网页日历应用：

- 一个月一页显示
- 支持上个月 / 下个月切换
- 支持跳转到指定月份
- 支持回到今天
- 自动高亮今天
- 支持自定义日期标记，可按星期几、每月几号或指定月份日期设置颜色
- 支持批量多选标记条件，可全选、反选、清空
- 自定义标记会长期保存到浏览器本地，下次打开或刷新仍然保留
- 支持一键重置全部自定义标记
- 总览界面保持简洁，只显示日期数字
- 点击日期可查看详细信息，包括星期、季度、年内第几天、节日、生肖、星座等
- 支持 Esc 关闭详情弹窗
- 支持键盘左右方向键切换月份
- 手机和电脑均可使用
